# pep2pro
easy pep2pro - from peptide to proteome V2

## Prerequistis

Following python libraries are required:

- json
- click
- configparser
- urllib2
- cookielib
- msparser
- Bio

If you want to import the data into a MySQL Database:

- mysql.connector
